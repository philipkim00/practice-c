사용자의 이름을 입력받는것.

결국 특정한 규칙과 패턴을 따른다.

CS50 Sandbox에서는 스크래치의 ask함수와 가장 비슷한 것은 get_string 함수입니다.

string(문자열이라는걸 알려줘야함) answer(변수)=get_string("What's your name?\n");

변수를 지정해서 물어볼수있게한다.

문자열은 문자열이라는 걸 string(형식지정자) 라고알려줘야함.

x=y 수학에서는 같다는 뜻이지만
c에서는 = 은 지정한다는 뜻 할당 연산자 오른쪽에 잇는 내용을 왼쪽으로 옮겨라.

%s=형식지정자
변수들을 지정할때 형식지정자를 사용한다.


$ clang -o string string.c -lcs50

여기서 -o string 은 string.c 를 string.out 이라는 머신코드로 저장하도록 하는 명령어입니다.

-lcs50은 “link”라는 의미를 지닌 -l 이라는 인자에 우리가 추가로 포함한 “cs50” 파일을 합친 것입니다. 이를 통해 컴파일시 cs50 파일을 연결하도록 알려줄 수 있습니다.


printf("hello, %s%s\n" , 변수1,변수2)

%s 처음거에 ,변수1 %s ,변수2 순서대로 지정된다.

문자열은 쌍따옴표 안에 들어간 0개 이상의 문자 배열이다.

clang은 하면 우리가 한 실수를 알려준다.

clang string.c 하면 a.out 라는 머신코드가 나오기때문에 이쁘게해주기위해
clang -o string string.c

-lcs50

-l 는 연결을의미  
 존나 쓸데없이 복작하므로

코드를 컴파일할떄 복작한 -o -lcs50을 외울필요없다
이런 명령어들을 외울필요없이 대신 개념만 이해하고 make string 해라.
make 명령어는 알아서 어떤 인자를 사용해야할지 알아서 한다.

make string.c가 아닌 make string 이름으로 만들어라 확장자 넣지말고

make 우리를 도와주는 똑똑한 명령어라 make hello 라고하면 hello.c 찾음
